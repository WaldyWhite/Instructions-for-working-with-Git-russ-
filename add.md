## <p style='text-align:center'>git add</p>
---
<br>

>Добавление **одного** файла в коммит.
```bash=
git add[имя_файла]
```
<br>

>Добавление **всех** файлов в коммит.

```bash=
git add .
```
<br>

>Команда позволит обновить отслеживаемые файлы.
```bash=
git add —update
```
<br>

 >А эта команда добавит даже **игнорируемые** файлы.
```bash=
git add —force 
```
